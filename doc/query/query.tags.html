标签伪类，不推荐使用，只是为了兼容jQuery。

  • :header匹配h1,h2,h3,h4,h5,h6。
  • :button匹配button与<input type="button"/>。
  • :link 匹配链接(a标签)
  • :text 匹配<input type="text"/>
  • :radio 匹配<input type="radio"/>
  • :checkbox 匹配<input type="checkbox"/>
  • :file 匹配<input type="file"/>
  • :password 匹配<input type="password"/>
  • :submit 匹配<input type="submit"/>
  • :image 匹配<input type="image"/>
  • :reset 匹配<input type="reset"/>
  • :input 匹配所有input标签
例子
请点击下面表格
$(":text").css("bgc","blue");
$(":radio").css("border","1px solid green");
$(":checkbox").css("border","1px solid red");
$(":button").css("bgc","#FF6363");
