$.fn.innerHeight( )

描述：

取得第一个元素的当前高度,包括padding，但是不包括border。

此方法是个只读文法。

此方法不适用于window 与 document对象，可以使用.height()代替。

w3c的盒子模型

例子 
    
    
 
 
             $.require("ready,node,css",function(){
                   var a = $(".innerHeight");
                   a.html("它的innerHeight为"+a.innerHeight()+"px");
               });
