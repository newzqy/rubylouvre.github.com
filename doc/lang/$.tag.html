$.tag(tag,inner)

描述：

用于提供了一种高效拼接多行HTML片断的方式，免去写<与>与结束标签之苦。

tag
必需。String。开标签除去<与>的部分。如果是文本节点请传入""。
stop
必需。String。标签里面的innerHTML，我们可以传入另一个$.tag作为它的参数。

返回值：

Object。想要得到HTML片断，需要用String()或+""处理一下。

例子 
                $.require("lang,node",function(){
                    var T = $.tag;
                    var html = T("P title=aaa",T("span","111111")("strong","22222"))("div",T("div",T("span","两层")))("H1",T("span","33333"))('',"这是纯文本");
                    $("js_tag_ex1).html(html)
                });
