$.String.pad(digits,[filling],[right] [radix])

描述：

在左边或右边补上一些字符。

digits
必需。Number。新字符的长度。
filling
可选。String。填空物，默认为"0"。
right
可选。布尔。不指明则往左边添加。
radix
可选。String。填空物，默认为10。
例子 
                $.require("lang",function(){
                    var a = $.lang("B");
                    $.log(a.pad(4,"0",0，16),true);
                });
