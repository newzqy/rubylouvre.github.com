$.String.stripTags( str )

描述：

去掉字符串中的html标签，但这方法有缺陷，如里面有script标签，会把这些不该显示出来的脚本也显示出来了

当前的实现：

stripTags: function(str) {
    return String(str || "").replace(/<[^>]+>/g, '');
}
例子 
$.require("lang", function(_) {
    var s = "<div>你好<script>alert(1)<\/script></div>";
    $.log(_(s).stripTags(), true);
});
