$.Array.diff(arr,arr2)

描述：

返回当前数组与arr2的差集,该数组包括了所有在当前数组但是不在arr2数组中的元素。它拥有一个别名函数叫without。

参数：

arr2
必需。Array。
例子 
                $.require("lang",function(){
                    var arr1 = [1,2,3,8];
                    var arr2 = [2,3,4];
                    var ret = $.Array.diff(arr1, arr2);
                    ret = $.dump(ret);//序列化
                    $.log(ret,true);
                });
