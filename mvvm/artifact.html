石器时代的舰队

我的第一代UI库，基于twitter的bootstrap发展而来。

每个UI都位于$.ui命名空间之下,所有UI都有以下通用之外

参数：

parent
可选。String|Element。指定UI组件要添加的位置，没有默认为body
preRender
可选。Function。当插入DOM树前执行的函数，只执行一次

实例的属性与方法：

data
用于保存经过补允与修正的用户传参，总是为一个对象
VM
根据上面的data转换过来的ViewModel对象，我们可以通过它修改视图的内容或样式
tmpl
当前组件所应用的模板
ui
当前组件的根节点

我最初的组件(不列入石器时代)

面板示例

它的调用代码，我们可以通过操作VM来实现控件的显隐与内容刷新。

$.require("ready,./more/artifact/panel", function() {
    var model = new $.ui.Panel()
    setTimeout(function() {
        model.hide()
    }, 1000)
    setTimeout(function() {
        model.VM.content.title("新标题")
        model.show()
    }, 2000)
})
