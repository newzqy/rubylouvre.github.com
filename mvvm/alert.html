alert

信息通知组件。

参数：

通知从外形上分为两种：普通通知与块状通知。块状通知比普通通知多了一个.alert-block，其padding更大，通常容纳文本也更多。另外，我们也可以通过.alert-heading来修饰标题。

通知从颜色上分为三种，成功、失败与正常的通知，对应的类名分别是alert-success，alert-error，alert-info。 但即便没这三个类名，组件本身就表现为"淡黄"的警示色，因此实际有四种颜色。

parent
可选。String|Element。指定UI组件要添加的位置，没有默认为body
preRender
可选。Function。当插入DOM树前执行的函数，只执行一次
text
可选。String。通知内容的主体
head
可选。String。仅在block参数为真正时可用。
cls
可选。String。控制其样式,可选值有alert-success，alert-error，alert-info。
block
可选。String。变成可添加更多内容外观更大方的块状通知，如果head长度不为零，会多出一个标题。

除了通用成员外，此实例还拥有以下成员：

bind
方法，从$.Flow继承而来。
fire
方法，从$.Flow继承而来。
unbind
方法，从$.Flow继承而来。
close(fn)
方法，手动关闭控件，从DOM树移除。如果它支持CSS3的渐变属性，还会有淡出动画。
fn如果存在并且为函数，将订阅成close事件的回调，回调的参数为当前实例。

事件：

通知类扩展了一组事件，可以介入通知的某些功能实现。

事件 描述
close 在调用 close方法时立刻触发。
closed 在通知已经关闭后（等待css过渡效果完成）触发。 
$.require("ready,./more/artifact/alert", function() {

    new $.ui.Alert('<a href="javascript:void(0);">有 3 条新微博，点击查看</a>', "alert-info", "fieldset");
    new $.ui.Alert("<strong>警告！</strong> 不要拿法律当挡箭牌。", " ", "fieldset");
    var ui = new $.ui.Alert({
        parent: "fieldset",
        block: 1,
        head: "微博控们注意了！",
        text: "<p>出游微博晒幸福时，最好不要泄露出游时间、人数，别泄露住址，谨慎晒出游照片，" + "以免让贼判断出你家是否有人。因为小偷说不定会盯上你的微博，“随着微博的发展，贼也很时尚啊！”</p>" + '<p><a class="btn btn-danger" href="#">接受建议</a> <a class="btn" href="#">不当回事</a></p>'
    });
    setTimeout(function() {
        ui.close(function(e) {
            alert(e.data.head)
        })
    }, 2500)

})
