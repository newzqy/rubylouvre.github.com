class绑定

用于为元素添加或移除指定的元素。

如果传参是一个对象，则遍历对象，将为真值的键名作为元素的新类名， 如果为假则移除此类名。如果传参为字符串，也作新类名添加给元素。

示例
同时加上三个类名

HTML：


<style>
    .my-class{
        color: yellow;
        background:orange;
    }
    .your-class {
        line-height:3;
        width:200px;
    }
    .his-class{
        text-indent:2em;
        font-size:14px;
    }
</style>
<div bind="class: { my-class: someValue,your-class: otherValue, his-class: moreValue }">
    同时加上三个类名
</div>

JS:

$.require("ready,./more/avalon", function() {
    var model = $.ViewModel({
        someValue: true,
        otherValue: true,
        moreValue: true
    });
    $.View(model);
})
