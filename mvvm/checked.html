checked绑定

示例
这由上面的元素checked状态控制显示的

HTML


   <label><input type="checkbox" bind="checked: displayMessage" /> 通过checkbox显示下面的DIV</label>
   <div bind="if: displayMessage">这由上面的元素checked状态控制显示的</div>

checked绑定要求它对应的值必须是函数。

JS

$.require("ready,./more/avalon", function() {
    var model = $.ViewModel({
        displayMessage: false
    });
    $.View(model);
    setTimeout(function() {
        model.displayMessage(true)
    }, 1000)

})
