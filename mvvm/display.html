display绑定

通过第一个传参的真假值决定其是否显示。

示例

XXXXXXXXXXXXXXXXXXXXXXXXXXXX

YYYYYYYYYYYYYYYYYYYYYYYYYYYYY

ZZZZZZZZZZZZZZZZZZZZZZZZZZZZZ

HTML：


    <p bind="display: show">
       XXXXXXXXXXXXXXXXXXXXXXXXXXXX
    </p>
    <p bind="display: function(){return 1}">
       YYYYYYYYYYYYYYYYYYYYYYYYYYYYY
    </p>
    <p bind="display:0">
       ZZZZZZZZZZZZZZZZZZZZZZZZZZZZZ
    </p>

JS：

$.require("ready,./more/avalon", function() {
    var model = $.ViewModel({
        show: true // 显示
    });
    $.View(model);
    setTimeout(function() {
        model.show(false); //隐藏
    }, 1000)
    setTimeout(function() {
        model.show(true); // 显示
    }, 2000)
})

一开始我们是看到XXXXXXXXXXXXXXXX，然后消失,再然后又出现。
