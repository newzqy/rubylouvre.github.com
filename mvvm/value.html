value绑定

第一个传参将强制转换为字符串，假值为""，然后变成目标节点的value属性,并同时对此元素进行input或propertychange监听。

示例

HTML：


   <input bind="value: content"><br/>
   <input bind="value: content">

JS。

$.require("ready,./more/avalon", function() {
    var model = $.ViewModel({
        content: "司徒正美"
    });
    $.View(model);
})
