style绑定

style绑定的参数总是一个对象，里面是它要添加的样式名与样式值。

相对于knockoutjs，它对键名不要求加引号，即使它是关键字，或是连字符风格，并且对于css3的新增属性， 它会自动转换为可用的形式， 如column-gap在webkit系浏览器自动转换为-webkit-column-gap，firefox下转换为-moz-column-gap。

示例
添加样式

HTML：


    <div bind="style: {
         box-shadow: '5px 5px 7px #888',
         background-color:'orange',
         color: 'purple',
         text-indent:currentProfit() < 0 ? '2em' : '4em',
         line-height: 3,
         width:'200px' }">添加样式</div>

JS:

$.require("ready,./more/avalon", function() {
    var model = $.ViewModel({
        currentProfit: 1500
    });
    $.View(model);
    setTimeout(function() {
        model.currentProfit(-70)
    }, 1500)
})
